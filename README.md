# HR Analytics Dashboard 📊

## Project Overview
This Power BI project provides a comprehensive analysis of employee data to help HR departments make data-driven decisions. The dashboard focuses on three main pillars: **Retention, Diversity, and Compensation.**

## 🖼️ Dashboard Preview
![HR Dashboard](https://github.com/samah21moh-boop/image/blob/main/HR_Dashboard.png)

## 🎯 Project Objectives
- Identify patterns and root causes of **Employee Attrition**.
- Analyze **Salary Distribution** across different departments and job roles.
- Monitor **Workforce Diversity** (Gender Ratio and Education levels).
- Calculate real-time KPIs for organizational health.

## 🛠️ Tools & Technologies
- **Power BI:** Data Visualization & Dashboarding.
- **DAX (Data Analysis Expressions):** Created advanced measures for Attrition Rate, Active Headcount, and Gender Ratios.
- **SQL:** Data querying and preparation.
- **Excel:** Initial data source and cleaning.

## 📈 Key Metrics Created
- **Attrition Rate:** `(Exited Employees / Total Employees) * 100`
- **Active Employees:** Filtered headcount for current staff.
- **Gender Ratio:** Breakdown of workforce diversity.

## 💡 Key Insights & Outcomes
- Provided actionable insights to optimize employee retention strategies.
- Identified budget allocation gaps across departments through salary analysis.
- Enhanced visibility into departmental performance and turnover risks.
